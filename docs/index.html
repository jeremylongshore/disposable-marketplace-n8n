Disposable Marketplace

Instant marketplace creation for quote collection from hundreds of resellers

Upload a CSV of reseller contacts and automatically collect ranked offers for luxury goods (watches, cars, art) within minutes. Dual outreach via email and API with automated ranking.

View on GitHub Quick Start Guide

Workflow Overview

Create temporary marketplaces on-demand for collecting competitive quotes from your reseller network. The workflow handles outreach, offer collection, ranking, and summary delivery automatically.

Workflow Diagram

graph LR
    A[Upload CSV] --> B[Validate Data]
    B --> C[Batch Processing]
    C --> D[Dual Outreach]
    D --> E[Email]
    D --> F[API]
    E --> G[Collect Offers]
    F --> G
    G --> H[Ranking System]
    H --> I[Top 5 Results]

Key Features

CSV-Based Setup

Upload reseller list with contact info, trust scores, and API endpoints. Workflow validates and processes automatically.

Dual Outreach

Simultaneously sends quote requests via email and API calls. Maximizes response rate from your network.

Automated Ranking

Scores offers based on price, trust, response time, and payment terms. Returns top 5 ranked results.

Quick Start

Prerequisites

  • n8n instance with webhook access
  • Google Sheets API credentials
  • SMTP credentials for email outreach
  • CSV file with reseller contacts

Installation

wget https://raw.githubusercontent.com/jeremylongshore/disposable-marketplace-n8n/main/workflow.json

Import into n8n and configure:

  • Webhook endpoints
  • Google Sheets integration
  • SMTP settings
  • Timeout values (default: 60 minutes)

Use Cases

  • Luxury watches (Rolex, Patek Philippe, Omega)
  • Classic cars (Ferrari, Porsche, vintage models)
  • Art & collectibles (paintings, rare books, wine)
  • Real estate (commercial properties, land)
  • Industrial equipment (manufacturing machinery)
